# 🎯 LibreChat Shared RAG 실제 테스트 가이드

## 현재 실행 상태 ✅

- **MongoDB**: localhost:27017 (정상 실행)
- **LibreChat**: http://localhost:3080 (정상 실행)
- **Shared RAG**: 모든 기능 구현 완료

---

## 📝 테스트 시나리오

### 1단계: 관리자 계정 생성

1. **브라우저에서 접속**: http://localhost:3080
2. **회원가입** 클릭
3. **관리자 이메일로 가입**:
   - 📧 이메일: `admin@example.com`
   - 👤 이름: `관리자`
   - 🔑 비밀번호: `your-password`

> ✨ **중요**: `admin@example.com`으로 가입하면 **자동으로 관리자 권한**이 부여됩니다!

### 2단계: 관리자 - Shared RAG 파일 업로드

1. **로그인 후 URL 이동**: http://localhost:3080/d/shared-rag
2. **관리자 전용 파일 관리 페이지** 확인
3. **테스트 파일 업로드**:
   - 파일 선택: `demo-company-policy.txt` (이미 생성됨)
   - 또는 새 텍스트 파일 생성하여 업로드
4. **업로드 확인**: 파일 목록에서 `✓ 텍스트 추출됨` 상태 확인

### 3단계: 일반 사용자 계정 생성

1. **로그아웃** 후 새 계정 생성
2. **일반 사용자로 가입**:
   - 📧 이메일: `user@example.com` (또는 다른 이메일)
   - 👤 이름: `일반사용자`
   - 🔑 비밀번호: `your-password`

### 4단계: Shared RAG 테스트

일반 사용자로 로그인하여 다음 질문들을 시도하세요:

#### 🔍 **테스트 질문들**:

```
휴가 신청은 어떻게 하나요?
```
**기대 답변**: "휴가 신청은 최소 7일 전에 상급자에게 제출해야 합니다..."

```
출장비 신청 절차가 어떻게 되나요?
```
**기대 답변**: "출장비 신청은 출장 완료 후 7일 이내에 제출해야 합니다..."

```
근무시간이 어떻게 되나요?
```
**기대 답변**: "정규 근무시간은 오전 9시부터 오후 6시까지입니다..."

```
교육비 지원 제도는 어떻게 되나요?
```
**기대 답변**: "교육비 지원은 연간 100만원 한도로 제공됩니다..."

## 🛠️ **문제 해결**

### MongoDB 재시작
```bash
sudo -u mongodb mongod --dbpath /data/db --logpath /var/log/mongodb/mongod.log --fork
```

### LibreChat 재시작
```bash
cd /workspace
npm run backend
```

### 로그 확인
```bash
# MongoDB 로그
sudo tail -f /var/log/mongodb/mongod.log

# LibreChat 로그 (백그라운드에서 실행 중이라면)
ps aux | grep node
```
