LibreChat RAG 시스템 사용 가이드

1. 개요
LibreChat은 오픈소스 AI 채팅 플랫폼으로, 다양한 AI 모델과 연동할 수 있는 기능을 제공합니다. 
이번에 추가된 RAG(Retrieval-Augmented Generation) 시스템을 통해 관리자가 업로드한 문서를 기반으로 
더욱 정확하고 맞춤형 답변을 제공할 수 있게 되었습니다.

2. 관리자 기능
관리자는 다음과 같은 기능을 사용할 수 있습니다:
- PDF, TXT, DOC, DOCX 파일 업로드
- 업로드된 파일 목록 조회 및 관리
- 파일 삭제 기능
- 문서 내용 검색 기능

관리자 계정은 하드코딩된 이메일 주소로 관리되며, 현재 다음 계정들이 관리자 권한을 가지고 있습니다:
- admin@librechat.com
- manager@librechat.com

3. 일반 사용자 기능
일반 사용자들은 관리자가 업로드한 문서를 기반으로 질문을 할 수 있습니다:
- 자연어로 질문 입력
- 관련 문서 청크 자동 검색
- 문서 기반 답변 생성
- 참조 소스 확인 가능

4. 기술적 구현
- MongoDB를 사용한 문서 청크 저장
- Langchain TextSplitter를 활용한 문서 분할
- 텍스트 기반 검색 인덱스 활용
- Express.js 기반 REST API
- React 기반 사용자 인터페이스

5. 보안 및 권한
- JWT 토큰 기반 인증
- 관리자 권한 미들웨어
- 파일 업로드 크기 제한 (10MB)
- 지원 파일 형식 제한

6. 사용 예시
질문: "LibreChat의 주요 기능은 무엇인가요?"
답변: 업로드된 문서를 기반으로 LibreChat의 AI 채팅 기능, RAG 시스템, 관리자 기능 등에 대한 답변을 제공합니다.

7. 향후 개선 계획
- 벡터 임베딩을 활용한 의미 기반 검색
- 더 다양한 파일 형식 지원
- 실시간 문서 업데이트 알림
- 사용자별 문서 접근 권한 관리
