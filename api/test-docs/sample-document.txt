회사 정책 가이드

1. 휴가 신청 절차
- 휴가 신청서를 작성하세요
- 최소 7일 전에 상급자에게 제출하세요
- 상급자 승인 후 인사팀 최종 승인 필요합니다

2. 출장 규정
- 국내 출장: 부서장 승인 필요
- 해외 출장: 임원 승인 필요
- 출장비 신청은 출장 완료 후 7일 이내 제출

3. 근무 시간
- 정규 근무시간: 오전 9시 - 오후 6시
- 점심시간: 오후 12시 - 오후 1시
- 유연근무제 적용 가능 (팀장 승인 필요)

4. 복지 혜택
- 건강보험 100% 회사 지원
- 연차 20일 (입사 첫해 15일)
- 교육비 지원: 연간 100만원 한도

5. 보안 정책
- 업무용 PC 개인 사용 금지
- 회사 자료 외부 반출 시 보안팀 승인 필요
- 비밀번호는 90일마다 변경
